# pattern-emitter-promise
Asynchronous pattern emitter using Promises

## Installation

Just do:
```
npm install pattern-emitter-promise
```
and you are ready.

This lib has no external dependencies.

## Usage

In your code:
```javascript
var EventEmitter = require('event-emitter-promise');
var ee = EventEmitter();

ee.on()

```
